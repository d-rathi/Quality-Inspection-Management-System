📚 Defect Library

Learn about common tire defects, their causes, impact and recommended actions.

🔵 Uniformity Defect

Description

Variation in tire uniformity that may cause vibration during driving.

Possible Causes

  • Material variation
  • Improper curing
  • Manufacturing inconsistency

Impact

Ride discomfort and vehicle vibration.

Recommended Action

Re-inspect the tire and verify machine settings.

🟣 Geometry Defect

Description

Incorrect tire geometry such as radial or lateral run-out.

Possible Causes

  • Improper molding
  • Machine calibration issue

Impact

Poor vehicle handling and uneven wear.

Recommended Action

Perform geometry inspection and calibration.

🟠 Balancing Defect

Description

Uneven weight distribution inside the tire.

Possible Causes

  • Material imbalance
  • Manufacturing variation

Impact

Steering vibration and poor driving stability.

Recommended Action

Perform dynamic balancing test.

🔴 Sidewall Defect

Description

Cracks, cuts or bulges on the tire sidewall.

Possible Causes

  • Handling damage
  • Manufacturing issue

Impact

Reduced tire strength and safety risk.

Recommended Action

Perform detailed visual inspection.

🟢 Bead Defect

Description

Damage near the bead area affecting rim fitting.

Possible Causes

  • Improper mounting
  • Manufacturing defect

Impact

Air leakage and sealing issues.

Recommended Action

Inspect bead area before approval.

🟤 Tread Defect

Description

Surface irregularities or uneven tread formation.

Possible Causes

  • Curing issue
  • Mold problem

Impact

Reduced traction and uneven wear.

Recommended Action

Verify curing and molding process.

⚫ Visual Defect

Description

Surface scratches, contamination or appearance defects.

Possible Causes

  • Handling issue
  • Surface contamination

Impact

Customer rejection and cosmetic quality issues.

Recommended Action

Clean and inspect before approval.

This library is intended for learning and reference purposes. It helps inspectors understand common tire defects before recording them in the system.
