# Quality Inspection Management System (QIMS)

## Overview
A web-based Quality Inspection Management System developed using Flask and SQLite for tyre manufacturing industries.

## Features
- User Login & Registration
- Dashboard
- New Inspection Entry
- Inspection Records
- Defect Tracker
- Defect Library
- Reports Dashboard
- Date-wise Report Filter
- Charts & Analytics
- PDF Report Generation

## Technologies Used
- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript
- Chart.js

## Future Enhancements
- Export to Excel
- Email Notifications
- Role-based Login
- Cloud Database Integration
